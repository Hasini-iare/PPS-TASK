## TASK-1: Student Performance Analyzer
This is a simple Python program that classifies student performance based on their marks.

🚀 Features

- Classifies performance into categories:

  - Average
  - Good
  - Excellent

## TASK-2: Crop Disease Risk Analyzer
This Python program calculates a risk score based on weather conditions and classifies it into risk levels.

🚀 Features

- Takes real-time input for:
  - Temperature 
  - Humidity
  - Rainfall
- Computes a weighted risk score
- Classifies risk into:
  - Low Risk
  - Medium Risk
  - High Risk
